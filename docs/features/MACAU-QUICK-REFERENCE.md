# 澳门演出功能 - 快速参考

## 功能说明
✅ **已上线** - 2026-07-22

当用户选择澳门的演出场馆时，系统自动提供珠海的交通和酒店信息。

## 为什么这样设计？

**澳门的特殊性：**
- ❌ 澳门没有民航机场（只有直升机场）
- ❌ 澳门没有高铁站
- 💰 澳门酒店价格较高，选择较少
- ✅ 珠海距离澳门很近，交通便利
- ✅ 珠海有机场、高铁站
- ✅ 珠海酒店选择多、性价比高

**用户最佳方案：**
飞到珠海 → 住珠海酒店 → 过关到澳门看演出

## 触发条件

场馆所在城市满足以下任一条件：
- 城市名包含"澳门"
- 城市名等于"Macau"
- 城市名等于"Macao"

## 实际效果

### 输入
- 场馆：澳门威尼斯人金光综艺馆
- 出发：北京
- 日期：2026-08-15

### 输出
- 🛫 机票：北京 → **珠海**
- 🚄 高铁：北京 → **珠海**
- 🚌 巴士：北京 → **珠海**
- 🏨 酒店：**珠海**市区

```
正在查询交通信息...
搜索机场和火车站（澳门演出，为您搜索珠海交通）
```

## 技术实现
- **判断逻辑**：检测场馆城市名
- **搜索切换**：机票/高铁/巴士搜索珠海
- **酒店特殊处理**：通过 POI 搜索珠海市中心位置
- **降级方案**：珠海信息获取失败时使用澳门场馆位置
