# 澳门演出珠海交通功能 - 完成总结

## 需求
当演出场馆为澳门的时候，提供珠海的酒店和机票信息。

## 实现方案

### 核心逻辑
1. **检测澳门场馆**：在获取场馆信息后，判断城市名是否包含"澳门"、"Macau"或"Macao"
2. **切换搜索目标**：将机票、高铁、巴士和酒店的搜索目标从澳门切换到珠海
3. **保存搜索位置**：将酒店搜索位置信息保存到状态中，供"加载更多"功能使用
4. **用户提示**：在加载时显示友好的提示信息

### 场景示例
**用户输入：**
- 艺人：张学友
- 出发城市：北京
- 演出场馆：澳门威尼斯人金光综艺馆

3. 搜索北京到珠海的机票
4. 搜索北京到珠海的高铁
5. 搜索北京到珠海的巴士

- 自动获取到珠海的交通信息（因为澳门没有机场和高铁站）

- 无需手动切换搜索城市
- 从珠海前往澳门的交通更便利
