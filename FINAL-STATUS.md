## 🔍 50001错误分析

### 可能的原因（按概率）

1. **offer_id已过期** - 60%
   - 有效期：10分钟
   - 从搜索到下单超时

2. **Token权限不足** - 30%
   - 测试Token可能没有创建订单权限
   - 需要联系龙虾确认

3. **航班状态变化** - 10%
   - 航班已售罄
   - 价格已变动
   - 航班已取消

## 🛠️ 技术人员操作指南

### 如何诊断50001错误

#### 1. 查看实时日志
```bash
wrangler tail
```

查找关键信息：
- `龙虾预订请求:` - 发送的数据
- `龙虾预订响应:` - 返回的错误
- `request_id` - 用于联系龙虾技术支持

#### 2. 验证Token权限

联系龙虾技术支持，提供：
- Token: `rdak_live_QmPWtfRKNBVyMcg07kn98sLPCMcwGmFk`
- Request ID: `85c8cd64-02f9-4785-b482-bb4c27833e55`
- 错误信息: `50001 - 机票下单失败`

询问：
- 这个Token是否有创建订单权限？
- 测试环境是否有特殊限制？
- 50001错误的具体含义？
