## 🔍 50001 错误的常见原因

### 1. offer_id 已过期 ⭐⭐⭐⭐⭐
**最常见原因（90%）**

- **有效期**：10 分钟
- **原因**：从搜索航班到点击购买间隔太久
- **解决**：重新搜索航班，立即购买

### 2. 航班已售罄 ⭐⭐
- 该舱位已没有座位
- 需要选择其他航班

### 3. 价格已变动 ⭐⭐
- 实时价格与搜索时不同
- 需要重新搜索

### 4. 航班时间问题 ⭐
- 航班已起飞或取消

## 🐛 如果还是 50001 错误

### 检查清单

- [ ] offer_id 是否在 10 分钟内？

1. **查看完整日志**
   ```bash
   wrangler tail
   ```

2. **查看 offer_id**
   - 在日志中找到 `"offer_id": "fs_xxx"`
   - 每次搜索会生成新的 offer_id

3. **测试其他航班**
   - 选择不同的航班
   - 选择不同的日期
